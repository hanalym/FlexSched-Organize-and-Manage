# Limitations and Future Improvements

## Current Limitations
- Only basic features are implemented.
- Console-based interface only (No GUI).
- Requirements are not saved permanently.
- Cannot support multiple users or accounts.
- No notifications or reminders outside the program.

## Future Improvements
- Create a Graphical User Interface (GUI).
- Save all requirements to a file or database.
- Implement task prioritization and sorting by deadlines.
- Add notifications or reminders, even if outside of program.
- Support multiple users with logins and personaized task lists.
