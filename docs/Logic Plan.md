```text
COMMENT: THIS WAS HEAVILY BASED ON FLEXSCHED SOURCE CODE 1



INITIATE THE PROGRAM

CREATE EMPTY LISTS OF REQUIREMENTS
CREATE EMPTY LISTS OF SCHEDULES
CREATE EMPTY LISTS OF NOTES


FUNCTION MENU
LOOP FOREVER
DISPLAY "FLEXSCHED MENU"
DISPLAY "1. NOTE REQUIREMENT"
DISPLAY "2. LIST DUE REQUIREMENTS"
DISPLAY "3. EDIT REQUIREMENT/DUE DATE/NOTE"
DISPLAY "4. REMOVE REQUIREMENT"
DISPLAY "5. UPDATE SCHEDULE"
DISPLAY "6. VIEW SCHEDULE"
DISPLAY "7. END PROGRAM"


ASK USER TO CHOOSE OPTION

IF CHOICE = 1
  ASK USER FOR REQUIREMENT TITLE
  ASK USER FOR DEADLINE
  ASK USER FOR NOTE
  STORE TITLE AND DEADLINE IN REQUIREMENTS LIST, NOTE IN NOTES LIST
  DISPLAY "REQUIREMENT ADDED SUCCESSFULLY"

ELSE IF CHOICE = 2
  DISPLAY "REQUIREMENTS LIST"
  FOR EACH REQUIREMENT IN REQUIREMENTS
    DISPLAY NUMBER, TITLE, NOTE, AND DEADLINE
    ENDFOR

ELSE IF CHOICE = 3
  ASK USER WHAT TO CHANGE (REQUIREMENT, DEADLINE, NOTE)
  IF REQUIREMENT
    ASK USER WHAT SPECIFIC REQUIREMENT TO CHANGE
    ASK USER TO INPUT EDITED VERSION
    ADD EDITED VERSION TO REQUIREMENT LIST
    REMOVE UNEDITED VERSION FROM REQUIREMENTS LIST
  ELSE IF DUE DATE
    ASK USER WHAT SPECIFIC DEADLINE TO CHANGE
    ASK USER TO INPUT EDITED VERSION 
    ADD EDITED VERSION TO REQUIREMENT LIST
    REMOVE UNEDITED VERSION TO REQUIREMENTS LIST
  ELSE IF NOTE
     ASK USER WHAT SPECIFIC NOTE TO CHANGE
     ASK USER TO INPUT EDITED VERSION 
     ADD EDITED VERSION TO RNOTES LIST
     REMOVE UNEDITED VERSION TO REQUIREMENTS LIST

ELSE IF CHOICE = 4
  DISPLAY "FEATURE IN THE WORKS!"

ELSE IF CHOICE = 5
  DISPLAY "FEATURE IN THE WORKS!"

ELSE IF CHOICE = 6
  DISPLAY "FEATURE IN THE WORKS!"

ELSE IF CHOICE = 7
  DISPLAY "EXITING FLEXSCHED"
  EXIT LOOP

ELSE
DISPLAY "INVALID OPTION"
ENDIF
END LOOP
END FUNCTION


CALL MENU

END PROGRAM
